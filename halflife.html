Evidence Half-Life v1.0.0

Estimates when a meta-analytic conclusion will become obsolete.

The Model

Given a cumulative meta-analysis (studies ordered chronologically), the tool estimates four quantities:

  1. Effect drift rate: slope of the cumulative pooled effect over time (weighted linear regression of cumulative theta on year)
  2. Fragility distance: how many standard errors the current pooled effect is from the null (|theta/SE|). Low values mean the conclusion is close to flipping.
  3. Heterogeneity trend: whether tau² is increasing over time (expanding disagreement = accelerating obsolescence)
  4. Evidence velocity: estimated rate of new studies per year in the field

Half-Life Computation

The half-life is the projected number of years until the cumulative z-score crosses the significance threshold (|z| = 1.96), assuming the current drift rate continues and new studies arrive at the estimated velocity. If the drift is toward the null, the half-life is finite. If the drift is away from the null (strengthening evidence), the half-life is reported as "> 50 years" (stable).

Formally: at the current rate, each new study shifts the cumulative z by approximately drift_rate / (SE * sqrt(k+j)) per year. We solve for the time T such that:

|z_current + drift_rate * T / SE_projected(T)| = z_alpha

Limitations

  • Assumes linear drift extrapolation — real evidence evolution is nonlinear
  • Does not model paradigm shifts (e.g., new drug class replacing old one)
  • Heterogeneity trend uses simple linear regression on tau² estimates, which are noisy
  • Based on Shojania et al. (JAMA 2007) framework but extends with quantitative modeling
